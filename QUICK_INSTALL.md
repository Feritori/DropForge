# 🚀 DropForge — Быстрая установка (БЕЗ SSH)

## ⚡ Самый простой способ (только браузер)

### Что нужно:
1. ✅ VPS с установленным Ubuntu/Debian
2. ✅ Запущенные PHP + MySQL (или просто MySQL)
3. ✅ Файлы DropForge на сервере

### Шаг 1: Подготовка сервера (1 команда)

```bash
# Запустите setup_vps.sh ОДИН РАЗ через SSH:
sudo bash setup_vps.sh
```

Этот скрипт установит: PHP, Nginx, MySQL, Composer, SSL

### Шаг 2: Загрузите файлы

```bash
# После setup_vps.sh файлы нужно загрузить:
cd /var/www
scp -r DropForge/* root@ваш-сервер:/var/www/dropforge/
```

### Шаг 3: Откройте установщик в браузере 🎯

```
http://ваш-IP/install.php
```

**ВСЁ! Дальше только браузер — никаких SSH команд!**

## 📋 Что делает веб-установщик

| Шаг | Что происходит | Нужно ли SSH? |
|-----|---------------|---------------|
| 1. Добро пожаловать | Информация об установке | ❌ Нет |
| 2. Проверка требований | PHP, MySQL расширения | ❌ Нет |
| 3. База данных | Создаёт БД и пользователя MySQL | ❌ Нет |
| 4. Таблицы | Создаёт 24 таблицы + данные | ❌ Нет |
| 5. Настройки сайта | Название, URL, Steam API | ❌ Нет |
| 6. Администратор | Создаёт аккаунт админа | ❌ Нет |
| 7. Config.php | Автоматически создаёт файл | ❌ Нет |

**Итого: 0 SSH команд после загрузки файлов!**

## 🗄️ Режимы настройки базы данных

### 🚀 Простой режим (рекомендуется)
```
Используем root MySQL — один пароль, всё работает
```

Нужен только root пароль от MySQL. Установщик сам:
- Создаст базу данных
- Настроит всё под root

### 🔐 Продвинутый режим
```
Создаём отдельного пользователя для безопасности
```

Нужен root пароль + придумать имя/пароль для пользователя БД.

## ❓ Частые вопросы

### Q: У меня нет root пароля MySQL
**A:** По умолчанию root без пароля — оставьте поле пустым.

### Q: Не записался config.php
**A:** Установщик предложит скачать файл. Загрузите его в `config/config.php`.

### Q: Как удалить install.php после установки?
**A:** Через SFTP или:
```bash
sudo rm /var/www/dropforge/install.php
```

### Q: Можно ли установить без Nginx?
**A:** Да, установщик работает с Apache. Просто загрузите файлы и откройте install.php.

## 🎯 После установки

1. **Удалите install.php!** (безопасность)
2. Войдите в админку: `/admin/index.php`
3. Настройте платёжки
4. Добавьте кейсы и предметы
5. Настройте SSL: `sudo certbot --nginx -d ваш-домен.ru`

## 🔧 Если что-то пошло не так

```bash
# Проверить MySQL
sudo systemctl status mysql

# Проверить PHP
php -v
php -m | grep pdo_mysql

# Проверить Nginx
sudo nginx -t
sudo systemctl status nginx

# Логи
tail -f /var/log/nginx/error.log
tail -f /var/log/mysql/error.log
```
