# 🚀 Установка новых функций DropForge

## Что добавлено:
1. **Live Feed** — лента выигрышей внизу страницы
2. **Уведомления в реальном времени** — всплывающие уведомления
3. **Конвертер валют** — отображение курса USD/RUB в шапке

---

## Шаг 1: Обновление базы данных

Выполните SQL-файлы в порядке:

```bash
# 1. Таблица для live feed
mysql -u ваш_пользователь -p ваша_база_данных < public/sql/add_live_feed.sql

# 2. Таблицы для системы тикетов
mysql -u ваш_пользователь -p ваша_база_данных < public/sql/add_support_tickets.sql
```

Или через phpMyAdmin:
1. Откройте базу данных
2. Перейдите в SQL
3. Скопируйте и выполните содержимое каждого файла

## Шаг 3: Тестирование

### Live Feed:
1. Откройте любой кейс (нужно быть авторизованным)
2. После выпадения предмета, он появится в ленте внизу страницы
3. Лента обновляется автоматически каждые 10 секунд

### Конвертер валют:
1. В шапке сайта справа от баланса есть кнопка "💱 1$ = XXX₽"
2. Кликните для просмотра таблицы конвертации

### Система тикетов:
1. Перейдите на /support.php
2. Нажмите "Создать тикет"
3. Заполните тему, сообщение и приоритет
4. Отправьте форму
5. Тикет появится в списке

### Мобильная адаптация:
1. Откройте сайт на мобильном устройстве или в режиме разработчика (F12 → Mobile)
2. Проверьте работу гамбургер-меню
3. Проверьте отображение кейсов и ленты

## Возможные проблемы и решения

### Live Feed не отображается:
- Проверьте, что таблица `live_wins` создана
- Откройте консоль браузера (F12) и проверьте ошибки
- Убедитесь, что хотя бы один кейс был открыт

### Ошибка 500 при создании тикета:
- Проверьте, что таблицы `support_tickets` и `support_messages` созданы
- Проверьте логи PHP на наличие ошибок

### Мобильное меню не работает:
- Проверьте, что CSS обновлён
- Очистите кэш браузера (Ctrl+F5)

## Для разработчиков

### Обновление live feed вручную:
```sql
SELECT * FROM live_wins ORDER BY created_at DESC LIMIT 10;
```

### Управление тикетами:
```sql
-- Все открытые тикеты
SELECT * FROM support_tickets WHERE status = 'open';

-- Закрыть тикет
UPDATE support_tickets SET status = 'closed' WHERE id = 1;

-- Сообщения тикета
SELECT * FROM support_messages WHERE ticket_id = 1 ORDER BY created_at ASC;
```
